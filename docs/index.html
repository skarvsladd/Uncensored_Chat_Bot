What is included

Chat completion examples

Small Python scripts demonstrate one-off chat completion requests and terminal-based chat loops.

Local endpoint support

Point requests at a local OpenAI-compatible server such as a desktop model runner on port 1234.

Conversation memory

Memory scripts append user and assistant messages so each response can use previous turns.

Quick setup

  1. Install Python dependencies.
  2. Start a local OpenAI-compatible model server, or configure a provider API key.
  3. Run one of the example scripts from the repository root.
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app_local_memory.py

Safety and privacy notes

Keep API keys out of source code and browser-delivered files. Prefer environment variables for provider credentials, and only run models or prompts that comply with your local laws, platform rules, and usage policies.
